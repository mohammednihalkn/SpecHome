# SpecHome

A Django-based web application designed for managing and viewing product specifications seamlessly by company and model.

## Features
* **User Authentication:** Secure login and registration system for users and administrators.
* **Product Management:** Add and organize products based on specific companies and models.
* **Specifications Tracker:** Detailed technical specification parameters for each product model.
* **Clean Interface:** Responsive HTML templates for seamless viewing and data entry.

## Tech Stack
* **Backend:** Python, Django
* **Database:** SQLite3
* **Frontend:** HTML, CSS

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone [https://github.com/mohammednihalkn/SpecHome.git](https://github.com/mohammednihalkn/SpecHome.git)

2. Install dependencies:
   pip install -r requirements.txt

3. Run database migrations:
   python manage.py migrate

4. Start the development server:
   python manage.py runserver
